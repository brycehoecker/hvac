Command Line Options

-h: print this usage information.
-nodb: turn database logging off (on by default). Information is still logged to local text files in $VHVDIR/DBLogs.
-noauto: turn automatic response to high pixel currents off (on by default).
-recover: start vhv in recovery mode: pixel voltage and power are read from HV crates, rather than taken from default file. Useful when vhv crashes in the middle of a run.
-nocrates: disable connection to HV crates to run in a test mode.
-nocurrents: disable connection to current monitor to run in a test mode. Also disables automatic response to high pixel currents.
-hvfile "filename": read HV settings from "filename" rather than from database.
