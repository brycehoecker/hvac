HV Params Menu

Commands

Scale to... % Opens a dialog box in which you can enter a percentage by which to scale the voltages from their current values.
Offset by... V Opens a dialog box in which you can enter a voltage by which to offset all pixels.
Set All Pixels to... V Opens a dialog box in which you can enter a voltage to which to set all pixels.
Set AutoResponse Params Opens a dialog box for configuring how vhv responds to high pixel currents.
Flat FieldNot yet tested! Opens a dialog box that allows you to (in principle) flatfield the camera based on the PMT currents.

Usage

Setting the autoresponse parameters

By default, AutoResponse is turned on. It can be disabled at the command line via the "-noauto" flag or disabled in the AutoResponse dialog box by unchecking "Turn AutoResponse ON."

By default, "Restore when Stars Clear" is off -- checking this box will cause vhv to guess whether a nearby (within 0.25 deg) star is responsible for a pixel's high current, and if so to periodically check whether that star has moved far enough away to try restoring it. "Far enough away" is defined as the distance between the star and the pixel at the time the pixel suppresses plus a small safety margin set in the "Restore Margin" box.
