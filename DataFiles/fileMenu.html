File Menu

Commands

Read Voltages from DB: reads HV settings from the database.
Write Voltages to DB: writes HV settings to the database AND generates a text file named for the current date and time AND updates the link default.hv to point to that new file.
Load File: loads HV settings from a text file.
Save File: saves HV settings to a text file.
Exit: quits vhv.

Usage

On startup by default, vhv will read voltages from the database and start with the power off. You can override this behavior from the command line. The "-hvfile" option allows you to specify a text file to read from instead of the database (still starts with power off). The "-recover" option asks vhv to start up using the current voltage/power settings in the HV crates (useful for recovering from a vhv crash in the middle of a run!).
To permanently update the HV settings Select "Write to DB".
To save a set of HV values for a special test Select "Save File" and give an informative name to your file.
To read the default HV settings Select "Read from DB".
If the database is inaccessible on startup, before the main window opens you"ll get a dialog box asking you to open an HV file. Select your favorite -- note that default.hv is linked to a file that matches what"s in the database.

Also, vhv periodically writes the current settings to a file "now.hv" -- so if you have been adjusting voltages and vhv crashes, this file is a useful recovery point.
